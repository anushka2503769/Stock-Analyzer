# 📊 Stock Fundamental Analysis Report Generator

A professional-grade Python tool that generates comprehensive PDF equity research reports with charts, valuation models, sentiment analysis, and an AI-written conclusion.

---

## 🚀 Quick Start

### 1. Install dependencies
```bash
pip install -r requirements.txt
```

### 2. Run
```bash
python main.py
```

You'll be prompted to enter one or more tickers:
```
Tickers: AAPL, MSFT, NVDA
```

A PDF report is generated for each ticker in the `reports/` folder.

---

## 📋 What's in Each Report

| Section | Contents |
|---|---|
| **Cover Page** | Company overview, key metrics, 5Y price chart |
| **Company Overview** | Business description, HQ, employees, leadership |
| **Price & Technicals** | 1Y chart with MA50/MA200, Bollinger Bands, RSI, MACD, Volume, returns |
| **Valuation Metrics** | P/E, Forward P/E, P/B, P/S, EV/EBITDA, EV/FCF, PEG, Graham Number, dividend yield |
| **Financial Health** | Liquidity ratios, debt analysis, Altman Z-Score, Piotroski F-Score, margins |
| **Growth & Profitability** | Revenue/earnings growth, margin history, CAGR, scorecard heatmap |
| **DCF Valuation** | 5-year free cash flow DCF with margin-of-safety analysis |
| **Analyst Estimates** | Consensus, price targets (low/mean/median/high), upside % |
| **News Sentiment** | VADER sentiment on latest headlines, accuracy commentary |
| **AI Conclusion** | Plain-English investment summary generated by Claude |
| **Disclaimer** | Full risk disclosure |

---

## 🧮 Metrics Computed

**Valuation:** Trailing P/E, Forward P/E, PEG, P/B, P/S, EV/EBITDA, EV/Revenue, EV/FCF, Price/FCF, Graham Number, Dividend Yield, Payout Ratio

**Financial Health:** Current Ratio, Quick Ratio, Debt/Equity, Debt/EBITDA, Net Debt, Interest Coverage, Operating CF, FCF, ROE, ROA, Altman Z-Score, Piotroski F-Score (8-point)

**Growth:** Revenue CAGR (3Y), YoY revenue growth (2 years), earnings growth, quarterly growth

**Technicals:** RSI (14), MACD + Signal + Histogram, Bollinger Bands (20,2), MA50, MA200, ATR (14), Sharpe Ratio, Max Drawdown, Annualised Volatility, 1W/1M/6M/1Y returns

**Sentiment:** VADER compound score, positive/neutral/negative %, headline table

## 💡 Tips

- Run in **VS Code** with the integrated terminal
- For best results, use major US-listed tickers (e.g. AAPL, TSLA, NVDA, MSFT, JPM)
- Works with non-US stocks too: e.g. `9988.HK`, `ASML`, `SAP`
- ETFs will have limited fundamental data but price/technical sections will still work
- The AI conclusion uses the Anthropic API if available, otherwise falls back to a rule-based summary
